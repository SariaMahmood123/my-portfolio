Brick Breaker Game:

This project is a simple "Brick Breaker" game implemented using Pygame. The game involves controlling a paddle to bounce a ball and break bricks. 2048 Game:

This project is a Pygame-based implementation of the 2048 game, where players combine numbered tiles to reach a target score. It includes a main menu, sound effects, and a high score tracking system that saves scores to a file. The game handles user input for moving tiles, checking win/lose conditions, and displaying appropriate endgame messages.
